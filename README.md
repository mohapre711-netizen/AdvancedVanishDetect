
# Advanced Vanish Detect

مود Fabric 1.21.4 لكشف الـ vanish بطرق أذكى.

المميزات:
- كشف اختفاء اللاعبين من الـ tab
- مقارنة tab list مع entities بالعالم
- تنبيه صوتي
- رسائل كشف داخل اللعبة
- كشف hidden entities

طريقة البناء:
1- افتح المشروع بـ IntelliJ
2- شغل:
   gradlew build

الملف النهائي:
build/libs/advancedvanishdetect-1.0.0.jar
